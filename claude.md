This is a deadsimple ipad game served as a progressive web app.
The game is intended to help toddlers add numbers.

Gameplay:
1. A monkey has to cross a bridge to go from one side of the river to another
2. There are 5 "tiles" required to make the bridge
3. The game randomly initializes with sets of tiles that sum up to 5. Eg: (3,2) or (1,1,1,3) or (1, 4) and so on.
4. Each set can be drag dropped into the empty slot
5. Once the bridge is complete the monkey crosses the bridge and reaches the other side. The monkey jumps up and down with happiness at having crossed.
6. Afer each game, an again button pops up to reinitialize the game

## Development Environment

To run the development environment:
1. Run `http-server` 
2. Navigate to `localhost:8080`
